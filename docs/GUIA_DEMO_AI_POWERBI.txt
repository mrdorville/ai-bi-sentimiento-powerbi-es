GUIA RAPIDA DEMO – Power BI + Azure AI

1) ABRIR POWER BI DESKTOP Y CARGAR EL DATASET
   - Archivo: data/resenas_es_500_ascii.csv

2) CREAR PARÁMETROS EN POWER QUERY
   - EndpointTextAnalytics (Texto)
   - ApiKeyTextAnalytics (Texto)

3) CREAR LA FUNCIÓN PERSONALIZADA
   - Nombre: fxSentiment
   - Copiar y pegar el bloque M desde scripts/fxSentiment_anonimo.txt

4) APLICAR fxSentiment A LA COLUMNA [comentario]
   - Invocar la función fxSentiment sobre la columna de texto
   - Expandir los campos:
       • sentiment
       • confidenceScores.positive
       • confidenceScores.neutral
       • confidenceScores.negative

5) CREAR MEDIDAS DAX PRINCIPALES
   - Reseñas =
       COUNTROWS('resenas')

   - Positivas % =
       DIVIDE(
           CALCULATE(COUNTROWS('resenas'), 'resenas'[sentiment] = "positive"),
           [Reseñas]
       )

   - Negativas % =
       DIVIDE(
           CALCULATE(COUNTROWS('resenas'), 'resenas'[sentiment] = "negative"),
           [Reseñas]
       )

   - Sentimiento Promedio =
       AVERAGE('resenas'[positive]) - AVERAGE('resenas'[negative])

6) VISUALES RECOMENDADOS
   - Gráfico de barras por tipo de sentimiento y canal
   - Línea de tendencia por fecha (score promedio)
   - Tarjetas KPI con % positivas, % negativas y total de reseñas

7) BACKUP (RECOMENDADO)
   - Guardar una versión del PBIX con los resultados ya calculados
     por si el endpoint de Azure AI falla durante la presentación.
